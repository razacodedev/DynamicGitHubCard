How it works

Click Open GitHub Modal, enter a GitHub username and click Update Card. Use JS to call https://api.github.com/users/<username>, parse the JSON and update elements.

JS Hooks (IDs to use):

  • openModalBtn — button that opens modal
  • modalBackdrop — modal backdrop element
  • githubUsername — input field
  • submitBtn — button to fetch data
  • profileImage, name, Username, bio, followers, repos, gists

Enter GitHub Username

Type a username (e.g. octocat) and click Update Card.

⚠️ Enter username to get data

By clicking update, your JS will call GitHub's public API and populate the card above.
